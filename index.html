Music — Audio Covert Channel

Welcome to Jack’s Audio Covert Channel. This channel hides data inside an ultrasonic audio signal (19 kHz & 19.5 kHz). Uploading audio files here allows participants to exchange covert messages that are inaudible to humans.

How It Works

  • Your portion of the message is converted into bits.
  • Each bit is represented as a high-frequency tone:
    • 19,000 Hz = 0
    • 19,500 Hz = 1
  • The audio file also contains a shared CRC code used to match the three channels together.
